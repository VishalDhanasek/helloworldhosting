3.Image tag

  • Image tag is used to display image in webpage with size specifications.

    • SYNTAX: <img src="image name.format" alt="text" width=" " height=" ">
    Note:
    • src = File path with ectension
    • alt = image not found
    • width = size in px or em or vw
    • height = size in px or em or vw
